# META — PSY–SUD neuroanatomical similarity

Code and data accompanying the study on macroscale neuroanatomical similarity
between psychiatric disorders (PSY) and substance use disorders (SUD), based on
ENIGMA-derived cortical and subcortical effect-size (Cohen's *d*) maps.

The analysis quantifies pairwise PSY–SUD morphometric similarity, localizes
regional convergence, tests its alignment with cortical transcriptional,
functional, and microstructural organization, and relates the resulting
similarity hierarchy to developmental timing, epidemiological comorbidity, and
genetic correlation.

---

## Repository structure

```
META/
├── data/
│   └── raw/                      # all input data (see "Input data" below)
├── src/                          # analysis scripts, grouped by research question
├── ALL_outputs_RQ1/              # generated by the RQ1 similarity pipeline (created on first run)
├── results/                      # generated figures / CSVs (created on first run)
└── README.md
```

All scripts resolve their paths **relative to their own location** (via
`__file__`), so the repository runs from any location once cloned — no paths
need to be edited.

## Requirements

Python 3.10+ is recommended. Install the dependencies with:

```bash
pip install -r requirements.txt
```

or manually:

```bash
pip install numpy pandas scipy scikit-learn statsmodels matplotlib seaborn h5py tqdm adjustText brainsmash enigmatoolbox
```

Package notes:

- **enigmatoolbox** — provides ENIGMA cortical/subcortical plotting, the
  Desikan–Killiany parcellation utilities, and `fetch_ahba()`. Follow the
  official ENIGMA Toolbox installation instructions if `pip` alone does not
  resolve it.
- **brainsmash** — spatial-autocorrelation-preserving surrogate maps.
- **adjustText** — used only for label placement in some figures.

## Input data (`data/raw/`)

| File | Description |
|------|-------------|
| `PSY_adults.xlsx` | Adult psychiatric Cohen's *d* maps (68 cortical + 14 subcortical rows × disorders) |
| `PSY_adults_ctx.xlsx` | Adult cortex-only disorders (no subcortical data) |
| `PSY_adolescents.xlsx` | Pediatric psychiatric maps (cortex + subcortex) |
| `PSY_adolescents_ctx.xlsx` | Pediatric cortex-only disorders |
| `*_pvalue.xlsx` | Corresponding per-region p-value tables |
| `SUD.xlsx` | Substance use disorder Cohen's *d* maps (ALC, ATS, CAN, COC, NIC, OPI, and transdiagnostic SUD) |
| `PSY_age_of_onset.xlsx` | Peak / median / IQR age of onset per disorder |
| `age_onset_prevalence_of_disorders.xlsx` | Comorbidity prevalence table |
| `SUD_general_prevalence.xlsx` | Marginal SUD prevalence (for attributable-risk computation) |
| `PSY_SUD_genetic_corr.xlsx` | Cross-disorder genetic correlation estimates |
| `ahba_dme_scores_in_dk.csv` | AHBA diffusion-map-embedding transcriptional components (C1–C3), left hemisphere, DK atlas |
| `mica_hc100_gradient-FC.csv` | Functional-connectivity cortical gradient |
| `mica_hc100_gradient-MPC.csv` | Microstructural-profile-covariance cortical gradient |
| `centroids_ctx_68.mat` | DK-68 cortical centroids (used for spin permutations and the BrainSMASH distance matrix) |

All cortical maps use the Desikan–Killiany atlas (68 regions). The AHBA
transcriptional data are left-hemisphere only and are mirrored onto the right
hemisphere in the scripts that use them.

## Execution order

Some scripts consume the CSV outputs produced by the RQ1 similarity pipeline, so
run the pipeline first.

### 1. RQ0 — descriptive maps and demographics (optional, standalone)

```bash
python src/RQ0_all_cohend_maps.py         # cortical/subcortical Cohen's d maps
python src/RQ0_PSY_age_sex_barplots.py
python src/RQ0_SUD_agesex_barplots.py
python src/RQ0_barplot_cohend.py
```

### 2. RQ1 — similarity pipeline (**produces `ALL_outputs_RQ1/`**)

```bash
python src/RQ1_zscore_ctx_ptoz_pspin.py         # cortex similarity + spin nulls -> Z_/RAW_/PVAL_ csvs
python src/RQ1_zscore_ctx_ptoz_brainsmash.py    # cortex similarity + BrainSMASH nulls
python src/RQ1_raw_subctx.py                    # subcortex raw similarity
```

These must run before the figure/RQ3 scripts below, which read the generated
`Z_cortex_*.csv`, `RAW_cortex_*.csv`, `RAW_subctx_*.csv`, and `PVAL_cortex_*.csv`.

Then build the Figure 1 panels:

```bash
# Adults — spin-based significance only
python src/RQ1_figure1_cortex_updated_pspin.py

# Adults — BrainSMASH-based significance only
python src/RQ1_figure1_cortex_updated_brainsmash.py

# Adults — combined figure: asterisks only where significant under BOTH nulls (main figure)
python src/RQ1_figure1_cortex_updated_combined.py

# Pediatric — spin-based significance
python src/RQ1_figure1_cortex_updated_pediatrics.py

# Pediatric — BrainSMASH-based significance
python src/RQ1_figure1_cortex_updated_pediatrics_brainsmash.py
```

### 3. RQ2 — regional convergence and cortical organization

```bash
python src/RQ2_mean_PSYSUD.py             # mean PSY / SUD maps
python src/RQ2_shared_maps_pspin.py       # shared alteration map + spin test, LOO, clusters
python src/RQ2_shared_maps_brainsmash.py  # shared alteration map + BrainSMASH test
python src/RQ2_shared_PCA_corr.py         # shared map vs PC1 of the effect-size matrix
python src/RQ2_shared_without_AN.py       # sensitivity: shared map excluding AN & OCD
python src/RQ2_corr_C1C2C3.py             # shared map vs DME transcriptional components (C1–C3)
python src/RQ2_cluster_corr_C2C3.py       # cluster-stratified shared map vs C2/C3 (Supplementary Table S21)
python src/RQ2_corr_MPCFC.py              # shared map vs FC and MPC cortical gradients
```

### 4. RQ3 — external validation

```bash
python src/RQ3_ageonset_regression.py     # similarity vs peak age of onset (OLS on disorder-level means)
python src/RQ3_ageonset_mixedeffect.py    # sensitivity: mixed-effects model on pairwise observations
python src/RQ3_gen_com_corr.py            # similarity vs comorbidity (ARD) and genetic correlation
python src/RQ3_scatter_cluster.py         # per-cluster scatter / statistics
```

## Statistical conventions

- **Two-tailed permutation p-values.** Both the spin and BrainSMASH scripts
  compute two-tailed empirical p-values: the minimum of the upper and lower
  one-tailed counts is multiplied by 2 and capped at 1.0 before conversion to
  a z-score. Signed z-scores are derived from the one-tailed probability to
  preserve directionality, while the reported p-value is always two-tailed.
- **Spatial null models.** Two complementary null frameworks are used
  throughout: (i) spin-based spatial permutations (10,000 rotations of the
  cortical surface using the DK-68 centroids) and (ii) BrainSMASH surrogate maps
  (10,000 surrogates) with matched spatial autocorrelation, built from the real
  cortical centroid distance matrix.
- **FDR correction.** For the RQ1 similarity matrix, FDR-BH correction is
  applied within each SUD column separately, independently for the spin and
  BrainSMASH p-value matrices. For the RQ2 gradient analyses (C1, C2, C3, FC,
  MPC), FDR-BH correction is applied **jointly across all five gradient tests**
  within each null framework.
- **Significance criterion.** An association is reported as robust only when it
  survives FDR correction under **both** null frameworks simultaneously
  (`p_fdr_spin < 0.05` **and** `p_fdr_brainsmash < 0.05`). The combined figure
  (`RQ1_figure1_cortex_updated_combined.py`) implements this intersection
  criterion directly for the heatmap asterisks.
- **Reproducibility.** Random seeds are fixed (`np.random.seed(42)` and
  `seed=42` in BrainSMASH). Note that spin-based p-values in the fourth decimal
  place can vary slightly across BLAS/hardware configurations because of the
  random rotation generation; significance conclusions are stable.

## Notes / conventions

- Disorder clusters used across scripts:
  - **Psychotic:** SCZ, BD, CHR
  - **AN/OCD:** AN, OCD
  - **Mood/Anxiety:** MDD, PTSD
  - **Neurodevelopmental:** ASD, ADHD

  (Conduct disorder is not assigned to a cluster.)
- Similarity is quantified primarily with sign-inverted Euclidean distance;
  Spearman correlation and cosine similarity are computed as sensitivity metrics.
- Primary analyses are cortical; subcortical analyses are complementary.
